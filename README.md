# New Wheels – SQL Business Analysis

## Objective
Use SQL to analyze business performance data and generate quarterly reporting insights that support operational and strategic decision-making.

## Tools & Technologies
- SQL
- MySQL
- Joins
- Subqueries
- Window Functions
- Aggregations

## Approach
- Translated business questions into structured SQL queries
- Used joins to combine data across multiple tables
- Applied subqueries to answer layered business questions
- Leveraged window functions for ranking, rolling metrics, and advanced analysis
- Aggregated key metrics to build quarterly business reporting outputs

## Key Insights
- SQL queries revealed trends in business performance across reporting periods
- Window functions improved the ability to compare time-based metrics
- Results supported more informed quarterly planning and operational decisions

## Files
- `new_wheels_analysis.sql` – SQL queries used for business analysis and quarterly reporting
- `NewWheels_FinalProject.pdf` - Report containing query outputs, screenshots, and business insights
